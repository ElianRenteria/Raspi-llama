
# Raspi-llama

Raspi-llama is a desktop GUI for TinyLlama, a lightweight LLM chatbot designed to run on a Raspberry Pi. This project aims to provide an easy-to-use interface for interacting with the TinyLlama chatbot on small devices like Raspberry Pi.

## Features

- Desktop GUI built with Python
- Simple, lightweight chatbot integration
- Designed specifically for low-power devices like Raspberry Pi

## Requirements

Before running Raspi-llama, ensure that you have the following installed:

- Python 3.x
- Required dependencies (see `requirements.txt`)

## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/ElianRenteria/Raspi-llama.git
   cd Raspi-llama
   ```

2. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

1. Start the application by running the `main.py` file:

   ```bash
   python main.py
   ```

2. The GUI will launch, allowing you to interact with the TinyLlama chatbot.
